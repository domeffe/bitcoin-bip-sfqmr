# bitcoin-bip-sfqmr
SF-QMR: Bitcoin Quantum Mitigation Proposal. A phased transition to neutralize CRQC risks via freezing of Satoshi-era outputs and mandatory pre-2013 migration, with a 36 months ultimatum.. Unlike "full opt-in" plans, this is a "partial opt-in" proposal. Immutability exists only through resistance. Hashed post-2013 supply remains untouched.
